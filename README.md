# مودة ورحمة | Mawada & Rahma App 🌙

تطبيق ويب (PWA) متكامل مصمم للأسرة المسلمة، يهدف إلى تنظيم الوقت والمهام بناءً على مواقيت الصلاة الحقيقية، مع تعزيز السكينة الأسرية والوعي المالي والتربوي.

## ✨ المميزات الرئيسية
- 🌍 **مواقيت صلاة عالمية:** جلب آلي للمواقيت لأي مدينة ودولة عبر [Aladhan API](https://aladhan.com).
- 📅 **منظم أسبوعي ذكي:** إدارة مهام منفصلة لكل يوم من أيام الأسبوع السبعة.
- 💡 **محرك نصائح السكينة:** رسائل منبثقة متنوعة (زوجية، تربوية، اقتصادية، إيمانية).
- 📿 **عداد استغفار مدمج:** لتشجيع الأوراد اليومية وجلب البركة.
- 🔒 **خصوصية 100%:** جميع البيانات والمهام تُحفظ في متصفحك الشخصي فقط (Local Storage).
- 🌓 **وضع ليلي (Dark Mode):** مريح للعين في أوقات الفجر والليل.
- 📥 **نسخ احتياطي:** إمكانية تصدير واستيراد المهام لضمان عدم ضياعها.

## 🚀 كيف تبدأ؟
1. قم بتحميل ملف `index.html`.
2. افتحه في أي متصفح (Chrome, Safari, Firefox).
3. أدخل اسم مدينتك ودولتك واضغط **تحديث**.
4. (اختياري) اختر "إضافة إلى الشاشة الرئيسية" في جوالك ليعمل كتطبيق مستقل.

## 🛠 التكنولوجيا المستخدمة
- **HTML5 & CSS3:** تصميم عصري متجاوب (Mobile First).
- **JavaScript (Vanilla):** لإدارة المنطق والبيانات دون مكتبات خارجية.
- **REST API:** للاتصال بجداول المواقيت العالمية.
